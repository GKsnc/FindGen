    说明：输入商品ID，返回tuple；tuple元组中包含若干条（0，1，n）流通记录

2. declare_goods()

    说明：不实现；生产商生产商品，对网络进行宣告（发布区块？）

3. record()

    说明：不实现；一般记录生成程序的接口，输入参数参与者（私钥），商品ID（多个，如何简单输入多个？）；生成后进行广播；网络状况的检测
