记录的格式:  
流通内容：商品ID，时间戳，序号，流通标识  
认证内容：认证机制（参与者地址，Pubkey Script，签名）  
记录：版本+流通内容+认证内容  

商品ID（16进制int,是否混淆未定）；流通标识（生产，销售，购买，其他；暂定）；时间戳；其他未定
